que es html:

hyper text markup languaje.

es el lengujae estandar para crear paginas web.
describe la estructura
le indica al explorador que y como mostrarlo.
consiste de etiquetas.
las etiquetas indican parrafos, vinculos, etc.


como funciona:

como un arbol. en la que hay un nodo raiz, y de este derivan las demas.

que es una etiqueta:

<soy-una-etiqueta>
contenido aca va el contenido
</soy-una-etiqueta>



estructura de html:
<html>
<head>  //nos sirve para agregar mas descriciones.
       <tittle>
</head>

<body>
      <h1>   un heading o titulo   </h1>
       <p>   un parrafo   </p>
</body>

</html>
